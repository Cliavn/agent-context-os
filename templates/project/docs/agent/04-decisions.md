## ADR-0001：采用 Agent 协助开发操作系统

状态：accepted

日期：`<YYYY-MM-DD>`

背景：

项目需要让人类开发者和 AI Agent 共享上下文、业务语义、架构边界和质量门禁。

决策：

采用 `AGENTS.md` 作为入口，`docs/agent/` 作为协作系统，workflow、checklist、modules 和业务语义文档作为任务执行约束。

影响：

后续功能开发、Bug 修复、重构和评审都必须遵守本体系。涉及模块、字段、状态、接口、架构或质量流程变化时，必须同步更新对应文档。
