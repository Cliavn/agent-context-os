# Agent 自查规则

Agent 在提交最终结果前必须自查：

- 是否阅读了 `AGENTS.md` 和 `docs/agent/00-index.md`。
- 是否阅读了相关 workflow、checklist、模块文档和业务规则。
- 修改是否超出用户请求范围。
- 是否破坏模块职责或绕过公开接口。
- 是否处理错误路径和边界条件。
- 是否补充或更新测试。
- 是否评估文档同步需求。
- 是否说明验证结果和剩余风险。

## Review 输出顺序

如果用户要求 review，输出顺序为：

1. 问题清单，按严重程度排序。
2. 文件和行号引用。
3. 风险说明。
4. 测试缺口。
5. 简短总结。
