# Checklist：版本控制

- [ ] 已检查工作区状态。
- [ ] 已确认本次使用主工作区还是独立 Git worktree。
- [ ] 已区分本次改动和用户已有改动。
- [ ] 已运行必要验证。
- [ ] 只暂存本次相关文件。
- [ ] 提交信息清晰。
- [ ] 已创建本地 Git 提交，或已说明跳过原因。
- [ ] 如使用独立 Git worktree，已记录路径、分支、合并状态和清理状态。
- [ ] 如任务已合并或废弃，已执行 `git worktree remove <worktree路径>` 和 `git worktree prune`，或已说明不能清理的原因。
- [ ] 最终回复结尾包含本地提交状态和“是否推送”。
- [ ] 推送前已获得用户授权。
- [ ] 未执行破坏性命令清理用户改动。
