# Checklist：新增功能

- [ ] 已阅读变更分级、项目画像、项目风格画像、项目记忆、架构、技术栈和质量规则。
- [ ] 已判断变更等级，并确认是否需要升级。
- [ ] 已找到相关模块文档。
- [ ] 已找到相关业务语义文档。
- [ ] 如需求来自完整方案，已阅读方案摄取记录和系统影响矩阵。
- [ ] 如需求来自方案落实台账，已确认台账状态为 `confirmed` 或 `active`。
- [ ] 如需求来自方案落实台账，已确认本轮任务 ID、开发顺序、依赖和验收口径。
- [ ] 已明确输入、输出、错误路径和验收标准。
- [ ] 已确认非目标。
- [ ] 已复用现有模式和工具链。
- [ ] 已补充必要测试。
- [ ] 已运行相关验证。
- [ ] 如需求来自方案落实台账，已更新或报告对应任务状态。
- [ ] 如新增接口、字段、状态或配置，已更新文档。
- [ ] 已评估用户补充背景或新业务事实是否需要记忆回写。
- [ ] 已评估用户补充风格偏好或新端侧习惯是否需要写回项目风格画像。
- [ ] 如存在未确认信息，已记录开放问题或主动追问。
