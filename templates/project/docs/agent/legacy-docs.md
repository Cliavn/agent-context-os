# 旧文档索引

本文件用于记录 `<项目名>` 接入 Agent Context Engine 前已有的旧文档。

旧文档首次接入时只建立索引，不默认读取全文，也不默认作为当前事实。

## 状态说明

| 状态 | 含义 | 使用规则 |
|---|---|---|
| `unindexed` | 已发现但未整理 | 不作为上下文使用 |
| `indexed` | 已记录路径和主题 | 任务相关时可读取 |
| `reviewed` | 已对照代码或用户确认 | 可迁移有效事实 |
| `migrated` | 有效内容已写入当前文档 | 旧文档只用于回溯 |
| `deprecated` | 已确认过期 | 不作为实现依据 |

## 可信度说明

| 可信度 | 含义 |
|---|---|
| `unknown` | 未确认 |
| `low` | 疑似过期或缺少来源 |
| `medium` | 与部分代码或用户描述一致 |
| `high` | 已被代码、测试或用户明确确认 |

## 规则

- 首次接入阶段不得全量总结旧文档。
- 旧文档只有在任务需要时才读取全文。
- 旧文档内容必须经过代码、测试或用户确认后才能迁移为当前事实。
- 已迁移内容应写入 `memory.md`、模块文档、业务文档或历史问题库。
- 过期内容应标记为 `deprecated`，不得作为实现依据。
